# ErgoSense

A cross-platform desktop ergonomics assistant powered by AI.

## Windows Troubleshooting

### "Java" or "JavaScript" Error on Launch
If you encounter an error mentioning "java" or a JavaScript error when launching the application, it is likely due to missing system dependencies.

**Solution:**
1. Download and install the **Visual C++ Redistributable for Visual Studio 2015-2022**.
   - [Download Link (Microsoft)](https://aka.ms/vs/17/release/vc_redist.x64.exe)
2. Restart your computer.
3. Try launching ErgoSense again.

This application uses AI models that require these standard Windows libraries to function.
